# Responsive

Como hoy en dia los usuarios navegan desde diferentes dispositivos por lo que al crear una pagina web debemos de personalizar la vista de cada tipo de dispositivo desde donde el usaurio pueda acceder a la pagina.

Para hacer que la pagina se vea bien en todos los tipos de dispositivos o vistas, en vez de crear una pagina web para cada dispositivo, es usar el Responsive Web Design con los diferentes Media Quaries.


El Responsive Web Design se enfoca en adaptarse a las diferentes interacciones del usuario y a cada diferente resolucion (tamaño de pantalla), por lo que debera adaptarse a cada uno de la variedad de dispositivos y la herramienta para lograr esto son los media quaries los cuales se utilizan con la siguiente sintaxis.

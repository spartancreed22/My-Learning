# Colores

Los colores se pueden representar de bastantes formas (Hexadecimales - nombre -  hsl - rgb).

### Nombre
Se pueden usar los colores, solamente nombrandolos mediante el nombre apropiado, es decir:

color: blue;
color: black;
color: red;

Para esto se usan colores muy basicos y principales.

### Hexadecimal
En este caso hay que tener en cuenta que lo colores tendran valores numericos, pueden tener o 3 digitos o 6 digitos y al principio un #:

Por ejemplo el color negro se puede representar de las siguientes formas:
#000 o #000000.

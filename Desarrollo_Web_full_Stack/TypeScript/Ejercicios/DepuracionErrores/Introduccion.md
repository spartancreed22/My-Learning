# Qué veremos en esta sección

Vamos a aprender a depurar o corregir errores, además de conocer de mucha mejor forma nuestro archivo de tsconfig.json.

Nos enfocaremos en aprender:

    * Porque siempre se compila a Js
    * La funcionalidad completa del archivo de configuracion de ts
    * Hacer depuracion completa de errores de forma directa a nuestros archivos ts
    * Remover los comentarios de los archivos realizados en produccion
    * Le diremos (restringiremos) a nuestro compilador que solamente se enfoque en ciertos archivos o carpetas.
    * Crearemos un archivo final para la salida.
    * Como canviar la version de salida de Js

Adicionalmente tendrán el conocimiento necesario para compilar automáticamente cualquier archivo que se vaya creando al momento de ser insertado a nuestro proyecto.

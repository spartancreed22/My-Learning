# Instalacion.

Para instalar TSI tenemos que tener en cuenta que debemos instalarlo para poder usarlo como cualquier lenguage de programacion.

Nos dirigiremos a la pagina https://www.typescriptlang.org/ para ver las diferentes formas de instalacion que tenemos.

Podremos instalarlo solamente para le proyecto mediante npm o node, usando este comando.

    npm install typescript --save-dev
    
podemos tambien encontrar mas abajo el apartado para instalarlo de forma global, donde tendremos el comando para instalarlo mediante npm.

    npm install -g typescript

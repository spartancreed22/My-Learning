# Tecnologias del proyecto.

En este proyecto usaremos Next.js 15 + Prisma ORM + TanStack Query: Proyecto Full Stack.

Con next,js utilizando react 19 y todo mediante actions y cada una de ellas interactuan de forma directa con las bases de datos (con estas se aplica el CRUD).
Nuestras bases de datos estaran gestionadas con prisma para que su configuracion sea lo mas sencillo posible, ademas de todo lo anterior usaremos TanStack Query para el proceso de revalidacion de datos o de sus estados los cuales nos generan para poder interactuar con las acciones anteriormente mencionadas del servidor.

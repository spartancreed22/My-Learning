METODOS HTTP

Un protocolo es un conjunto de normas y reglas que son aplicadas para guiar un proceso o acción

HTTP: Permite que aquellas solicitudes (Request) y respuestas (Responses) entre el cliente y el servidor tengan un formato determinado que se debe seguir y respetar para que no existan errores al comunicarse

Una request o solicitud  que se realiza en un comunicación que utiliza el protocolo http tiene  una serie de partes, donde cada una de ellas cumple con una funcionalidad

#    *MARKDOWN*

Es un lenguaje de marcado de texto el cual permite convertir un texto plano a HTML y el objetivo es poder leer y escribir sin necesidad de conocer HTML y es usado con normalidad en documentación, blogs, comentarios en foros, y en muchas otras aplicaciones que requieren un formato sencillo y limpio.

    *Características*

- Simplificación: Es fácil de aprender y usar, para esto usa una sintaxis mínima y básica para convertirla en HTML
  
- Legibilidad: Estos textos son super fáciles de leer así sin mucho procesamiento adicional; esa es la intención, que el texto sea legible sin necesidad de se procesado
  
- Portabilidad: Es texto plano, lo que lo hace portátil y fácil de versionar y compartir.

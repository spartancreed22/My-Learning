# Matrices.

Las matrices son arreglos que tiene dos dimensiones divididos en columnas y renglones o filas

como son tablas si queremos dirigirnos a una en especifico entonces 
 

dependiendo del numero de filas y columnas entonces dependiendo de estas vamos a ubicarnos

Coordenadas:

    [0][0] [0][1] [0][2]
    [1][0] [1][1] [1][2]
    [2][0] [2][1] [2][2]

    los valores:

    5   8   1
    9   3   2
    4   7   6

    Con Bordes (3x3)


    +---+---+---+
    | 5 | 8 | 1 |
    +---+---+---+
    | 9 | 3 | 2 |
    +---+---+---+
    | 4 | 7 | 6 |
    +---+---+---+

### Codificacion: 

int[][] matriz = {
    {5, 8, 1},
    {9, 3, 2},
    {4, 7, 6}
};

definimos una matriz:

var matriz = new int[2][3];

    //aqui estamos creando la matriz teniendo en cuenta primero el numero de filas y después el numero de columnas

    //Para dar los valores, debemos de especificar bien las coordenadas
    matriz [0][0]= 100;
    matriz [0][1]= 200;
    matriz [0][2]= 300;
    matriz [1][0]= 400;
    matriz [1][1]= 500;
    matriz [1][2]= 600;



## Recorrer Matrices

vamos a definir las filas y columnas con unas constantes

final int filas = 3;
final int columns = 2;

las vamos a usar en la matriz para ponerlas

var matriz = new int[columns][filas];

// para recorrerlos tenemos que hacer dos for para que recorran filas y columnas, el primero hace las filas
for(int fils = 0; fils < filas; fils++){
    for(int cols = 0; cols < columns; cols ++){
        System.out.println("Coordenadas[" +fils + "] [" + cols + "] = "  + matriz[fils][cols]);
    }
};


Para crear una matriz de forma mas simplificada y de una vez asignar los valores se hará de la siguiente forma:

var matriz = new int[][]{
    {100, 200, 300},
    {400, 500, 600}

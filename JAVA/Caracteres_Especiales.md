# Caracteres especiales.
En java existen varios caracteres especiales en los cuales tenemos: 

"\n " = con este caracter haremos saltos de lineas.

"\t " = su funcion es realizar una tabulacion.

"\' " = este caracter se usa para poner una comilla simple en un texto.


"\" " = este caracter se usa para usar una comilla doble en el texto pero si la ponemos sin el backeslasu entonces cortaremos el texto arrojando error.

"\" " = Para poner una barra invertida la ponemos dos veces

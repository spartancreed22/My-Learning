# Java

## INTRODUCCIÓN

Es un lenguaje de alto nivel, es compilado y de proposito general.
Fue creado por James Gosling en 1995.
Es un lenguaje muy demandado en la industria y muy utilizado en diferentes campos

Razones: 
Amplitud y diversas aplicaciones
Es un lenguage omnipresente, se suele utilizar en varias aplicaciones como lo son: Desarrollo web - aplicaciones moviles - sistemas empresariales - Aplicaciones de escritorio.


### Características 

* La caracteristicas mas importantes de Java son: la portabilidad e independencia.
* Una de su mayor fortaleza es su lema: Escribe una vez y ejecuta en cualquier lugar.


1) Portabilidad e independencia de plataforma: Como decia su lema, que el codigo compilado de java se puede ejecutar en cualquier sistema operativo o plataforma que tenga instalado una maquina virtual de java.
   
2) Seguridad y rendimiento: este lenguage fue diseñado con la seguridad de una vez integrada, a que su arquitectura características que lo protegen contra una muy amplia gama de amenazas y vulnerabilidades.
   
3) Comunidad y Ecosistema: Java cuenta con una de las comunidades de desarrolladores las mas grandes, por lo tanto hay una basta cantidad de recursos, bibliotecas o frame works y herramientas disponibles para el desarrollo.
   
4) Oportunidades Profesionales: El mercado laboral de java es muy amplio, en crecimiento y desarrollada.
   
5) Innovaciones y Futuro: Java no es solo un lenguage sino tambien es una plataforma versatil y potente que a durado con el tiempo.


Primero para poder ejecutar los programas en java, necesitamos instalar algunas herramientas como el compilador y una maquina virtual de java.

Compilador: Es un programa el cual traduce el lenguage de java a un lenguage de bajo nivel (como codigo binario el cual maneja la maquina) el cual puede ser ejecutado por el sistema operativo

El compilador de java (javac) convierte el codigo fuente (.java) en codigo bytecode ( .class) 

### JDK

Es un paquete de herramientas necesarias para poder ejecutar y desarrollar aplicaciones JAVA,

* Compilador: traduce el codigo de Java a bytecode.
* Java Rutine Enviroment (JRE): (entorno de ejccion de java) Proporciona bibliotecas y la JVM (java virtual machine) necesarias para ejecutar codigo de bytecode.
* Herramientas de desarrollo: incluye herramientas como el depurador, el empaquetador entre otras.
